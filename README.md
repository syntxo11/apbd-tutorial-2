\# apbd-curso-2



\## Assignment



This is an easy console app for overseeing university equipment rentals.



\## What it accomplishes



\- supplies apparatus



\- retains users



\- leases apparatus



\- equipment returns



\- computes fines for overdue returns



\- displays equipment that is accessible



\- displays past-due rentals



\- generates basic reports



\## Organization



\- `Models` - domain entities like equipment, users, and rentals



\- `Services` - business operations for leasing, returning, and reporting



\- `Program.cs` - example situation



\## Design



I divided models from services to make the code clearer and easier to manage.



The foundational classes are `Equipment` and `User`, from which specific types derive.



\## Running Instructions



Launch the project in Rider or Visual Studio and execute it.
